# Hospital Management System

A web-based Hospital Management System developed using Flask, SQLite, HTML, CSS, Bootstrap, and Python.

## Features

* Admin Login

* Patient Management

  * Add Patient
  * View Patient
  * Edit Patient
  * Delete Patient
  * Search Patient

* Doctor Management

  * Add Doctor
  * View Doctor
  * Edit Doctor
  * Delete Doctor

* Appointment Management

  * Book Appointment
  * Approve Appointment
  * Complete Appointment
  * Cancel Appointment

* Dashboard Analytics

  * Total Patients
  * Total Doctors
  * Total Appointments

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* Bootstrap
* Jinja2

## How to Run

1. Clone the repository
2. Install requirements

```bash
pip install -r requirements.txt
```

3. Run application

```bash
python app.py
```

4. Open browser

```text
http://127.0.0.1:5000
```
